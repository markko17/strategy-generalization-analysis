## 📋 What is strategy-generalization-analysis?

This application helps you study how trading strategies perform over time. It checks their strength by running tests using past data and simulating future results. You will get insights into how reliable your strategies are. The app lets you save data to Excel and run portfolio-level tests. It works with cryptocurrencies and other trading markets.

You do not need to code to use this app. The tool runs on your Windows computer and guides you through analyzing strategies step-by-step.

## 🧭 Using the Application

The app has a simple interface split into four main parts:

- **Data Input**: Upload your historical trading data here.
- **Strategy Setup**: Choose and configure your trading strategies.
- **Analysis**: Run tests to check strategy strength and robustness.
- **Export**: Save your results to Excel for further review.

---

### How to run your first analysis

1. Prepare your data file. The app accepts CSV files with date, price, and trade information.

2. In the Data Input tab, click "Upload" and select your CSV file.

3. Go to the Strategy Setup tab. Pick from the available strategy templates or configure your own.

4. Click "Run Analysis". The app will perform walk-forward testing and simulations.

5. Check results in the Analysis tab. You will see charts and numbers showing strategy performance.

6. To save your data, go to Export and pick "Save as Excel". This creates a report you can open in Excel.
